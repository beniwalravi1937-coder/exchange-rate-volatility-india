# Exchange Rate Volatility in India

This project analyzes exchange rate volatility in India using RBI data and statistical analysis.

The study divides the exchange rate regime into three phases:

Phase 1: 1960–1990 (Fixed/controlled exchange rate)

Phase 2: 1991–2000 (Economic liberalization)

Phase 3: 2000–2024 (Globalized floating exchange rate)

Methods used:

• Log return calculation
• ANOVA analysis
• Bonferroni post-hoc test
• Macroeconomic factor analysis

Variables analyzed:

• Inflation rate
• GDP growth rate
• Trade deficit
